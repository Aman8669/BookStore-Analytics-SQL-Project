# 📚 SQL Bookstore Analytics Project

A complete PostgreSQL-based bookstore management and sales analytics system built using SQL.

## 🚀 Project Overview
This project demonstrates practical implementation of relational database concepts, data analysis queries, and business intelligence reporting using PostgreSQL.

The system manages:
- Books inventory
- Customer information
- Order management
- Sales analytics
- Revenue tracking

## 🛠 Tech Stack
- PostgreSQL
- SQL
- pgAdmin

## 📂 Project Files
- `books.csv`
- `customers.csv`
- `orders.csv`
- SQL schema and query scripts

## 🔑 Features Implemented
- Relational database schema design
- Primary & Foreign Key relationships
- Data import from CSV files
- Filtering and sorting queries
- Aggregate functions and analytics
- JOIN operations
- GROUP BY & HAVING clauses
- Revenue and inventory analysis

## 📊 Business Insights Generated
- Most frequently ordered books
- Top spending customers
- Genre-wise sales analysis
- Remaining inventory tracking
- Revenue calculations
- Customer order patterns

## 📌 Advanced SQL Concepts Used
- JOIN

- HAVING
- ORDER BY
- LIMIT
- Aggregate Functions
- COALESCE
- DISTINCT
- Foreign Key Constraints
